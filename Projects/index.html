GMS Hintergrund Tool: (Nicht Fertig!)

Ich habe ein Tool entwickelt, das allen Leuten ermöglicht,
ihre eigenen Hintergrundbilder auf den Schulcomputern zu verwenden.
Denn nachdem die Schule im Frühjahr 2023 die Möglichkeit deaktiviert hat,
eigene Hintergrundbilder festzulegen, habe ich eine Lösung entwickelt.
Mein Tool nutzt eine Batch-Datei namens Autorun.bat, die den Windows Cache regelmäßig aktualisiert und in unter 5 Minuten nach der Anmeldung,
automatisch ein neues, vorher ausgewähltes Hintergrundbild anzeigt.
Um das Tool zu installieren, muss lediglich ein Bild genommen werden, in einen Ordner gelegt werden und eine Datei mit Doppelklick ausgeführt werden.
